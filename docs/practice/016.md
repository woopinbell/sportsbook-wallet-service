# after-commit InvariantChecker

커밋된 연산마다 분개 쌍의 네 가지 무결성을 검증하는 after-commit 리스너를 둔다.

## 개요

`InvariantChecker`는 `OperationCommitted` 도메인 이벤트를 `@TransactionalEventListener(AFTER_COMMIT)`로 구독해, 성공적으로 커밋된 모든 wallet 연산에 대해 네 가지를 검증한다. DB unique 제약과 `pair()` 팩토리가 나쁜 쌍이 애초에 들어오지 못하게 막는 1차 방어이고, 이 리스너는 조용한 위반(직접 INSERT, 버그 있는 마이그레이션, 운영자의 fix-it SQL)이 눈에 안 띄게 지나가지 못하게 하는 2차 장치다.

## 작업 순서

1. `OperationCommitted` 이벤트 record를 둔다.
2. `WalletService.writePair`가 쌍 저장 직후 이벤트를 publish한다.
3. `InvariantChecker`가 AFTER_COMMIT에 네 가지를 검증한다.
4. Micrometer 카운터로 위반/검사 횟수를 노출한다.

## 작업 내역

### 검증 네 가지

리스너 시그니처(구독 phase)는 노출하고, 네 검사 본문은 비운다.

**책임:** 한 `operation_group_id`의 분개에 대해 — (1) 정확히 두 entry, (2) 차변 1 + 대변 1(둘 다 같은 side면 안 됨), (3) 차변 금액 − 대변 금액 = 0, (4) 두 entry 같은 통화. 위반 시 `wallet.invariant.violations` 증가 + ERROR 로그. 매 호출 `wallet.invariant.checks` 증가.

```java
@TransactionalEventListener(phase = TransactionPhase.AFTER_COMMIT)
public void verify(OperationCommitted event) {
  // TODO: checks.increment()
  // TODO: ledgerRepo.findByOperationGroupId(event.operationGroupId())로 쌍 로드
  // TODO: 쌍 크기 / side 분리 / net == 0 / 통화 일치 검사. 위반이면 violations.increment() + log.error
}
```

### AFTER_COMMIT 타이밍

`WalletService`는 `writePair()`에서 쌍을 저장한 직후 `OperationCommitted`를 publish한다. 이벤트는 감싸는 트랜잭션이 커밋될 때까지 버스에 머문다. AFTER_COMMIT phase 덕에 롤백된 시도(잔고 부족, 제약 race-loser, 일반 DataIntegrityViolation)에는 리스너가 fire하지 않는다.

위반을 발견해도 커밋 후라 되돌릴 수 없다. ERROR 로그와 `wallet.invariant.violations` 카운터로 운영자가 기존 Prometheus 알람 체인으로 페이징된다(ADR-0007). 이 리스너는 *예방이 아니라 안전망*이다 — 예방은 DB 제약과 `pair()`가 한다.

### repository 집계 메서드

리스너와 다음 reconciliation 커밋을 위해 세 메서드가 `LedgerEntryRepository`에 추가된다.

**책임:** `findByOperationGroupId`(알림받은 쌍 조회), `netSumAll`(시스템 전체 차변−대변, 정상 시 0), `netByAccountAndBucket`(계좌·bucket별 차변−대변). 집계 두 개는 파생 쿼리로 안 나오므로 `@Query`로 합산식을 쓴다.

## 검증

- `mvn -q -Dtest=IntegrityIntegrationTest test` — 정상 debit이 checks만 올리고, 주입한 drift가 violations를 올리는지(018). *Testcontainers(PostgreSQL+Redis)가 떠야 한다.*
- 감각 점검: 롤백되는 시도(잔고 부족 debit) 후 리스너가 fire *하지 않는가*(checks 불변) / 짝 없는 차변 하나를 주입하면 네 검사 중 어느 것이 위반을 올리는가.
