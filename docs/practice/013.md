# /internal/v1/wallet REST 표면 노출

시스템 내부 서비스가 호출하는 6개 endpoint와 요청/응답 DTO를 노출한다.

## 개요

`WalletController`는 시스템의 다른 서비스가 wallet에 들어오는 HTTP 입구다. 모든 경로가 `/internal/v1/wallet` 아래에 있다(ADR-0004). "internal" 세그먼트는 gateway가 외부 클라이언트 트래픽을 여기로 직접 전달하지 않음을 뜻한다. betting-service, settlement-service, admin-api, 그리고 gateway의 내부 read 호출만 온다.

## 작업 순서

1. 6개 endpoint를 `@RestController`에 매핑한다.
2. transaction endpoint는 `Idempotency-Key` 헤더를 받는다.
3. 요청/응답 DTO를 record로 둔다.
4. `@Valid`로 Bean Validation을 건다.

## 작업 내역

### endpoint

경로 표는 API 계약이라 노출한다. 핸들러 본문(명령 변환 + 서비스 호출)은 비운다.

| Method | Path | 비고 |
|--------|------|------|
| POST | `/internal/v1/wallet/accounts` | 계좌 생성/반환 |
| GET | `/internal/v1/wallet/accounts/{userId}/balance` | 잔고 조회 |
| POST | `/internal/v1/wallet/transactions/deposit` | Idempotency-Key 필수 |
| POST | `/internal/v1/wallet/transactions/withdraw` | Idempotency-Key 필수 |
| POST | `/internal/v1/wallet/transactions/debit` | Idempotency-Key 필수 |
| POST | `/internal/v1/wallet/transactions/credit` | Idempotency-Key 필수 |

**책임:** transaction endpoint는 `Idempotency-Key`를 평문 String 헤더로 받아 `IdempotencyKey.of(...)`로 변환한다. shared-protocol compact 생성자가 와이어 형태(printable ASCII, 128자 이하)를 검증하고, 위반은 `IllegalArgumentException`으로 떠 다음 커밋(014)의 핸들러가 400으로 매핑한다.

### DTO

요청/응답을 record로 둔다(`OpenAccountRequest`, `TransactionRequest`(userId, amount), `CreditRequest`(userId, amount, source), `AccountResponse`, `BalanceResponse`, `WalletOperationResponse`). Money/Currency/IdempotencyKey는 shared-protocol에서 오고, Jackson 바인딩이 거기 고정돼 이 API를 소비하는 서비스마다 JSON 형태가 동일하다. `@Valid @RequestBody`마다 Bean Validation이 돈다 — record 컴포넌트의 `@NotNull`이 최상위 필드 누락을 막고, 금액 양수/통화 일치 같은 깊은 검사는 도메인 mutator에 남아 014의 핸들러로 떠오른다.

> 순수 DTO record는 계약이므로 위 목록으로 노출하고 별도 blank를 두지 않는다. 컨트롤러는 오류 매핑 없이 얇게 유지한다.

## 검증

- `mvn -q -Dtest=WalletControllerTest test` — `@WebMvcTest` 슬라이스(015)가 각 endpoint의 라우팅/바인딩을 검증한다(컨테이너/Docker 불필요).
- 감각 점검: transaction endpoint를 `Idempotency-Key` 헤더 없이 호출하면 누락이 드러나는가 / `@Valid` 본문의 `@NotNull` 위반이 400으로 떨어지는가 / 컨트롤러에 try/catch 오류 매핑이 *없는가*(014로 위임).
